I. Installation
----------------------------------------------------------------------------------
Die Applikation wurde unter dem Webserver XAMPP erstellt und ist so vorbereitet   |
das es ohne weiteren Einstellungen auch mit MAMP funktioniert.                    |
                                                                                  |
Verwendete PHP Version:     7.0.9                                                 |
Verwendete Datenbank:       MySQL                                                 |
DB Dump:                    pss_order_sheet.sql                                   |
----------------------------------------------------------------------------------

II. Funktionsbeschreibung
----------------------------------------------------------------------------------
Die Applikation ersetzt ein Formular zum Erfassen der internen Aufträge aus dem   |
Servicecenter.                                                                    |
Mit dem Formular werden dem Mitarbeiter im Servicecenter Informationen            |
für eine Neuinstallation/Übernahme eines PC/Notebooks bereitgestellt und          |
Verwaltet.                                                                        |
